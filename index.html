Fractol

This project explores fractals, abstract mathematical objects that exhibit self-similar patterns at every scale. Implemented using the MiniLibX graphical library, it allows visualization of both the Julia set and the Mandelbrot set. Key features include infinite zoom functionality via mouse wheel, dynamic parameterization for Julia sets, and interactive display of fractal types based on command-line input. The project enhances understanding of fractal geometry and graphical programming skills.
